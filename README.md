# Avatar Tinker Vista - ApplyOnPlayHack

# What was this?
This was a hacked alternative implementation of Apply on Play of NDMF that maintains NDMF Platforms of avatars when Apply On Play is performed.

The implementation is basically the same as Apply on Play of [Ablet](https://github.com/kaikoga/Ablet-Unity), so migration to Ablet is recommended as long as a similar behavior is desired. 

# これはなに？

複数のNDMFプラットフォームを扱うプロジェクトで、 Apply on Playの際にアバターのプラットフォームを維持するApply on Playの実装でした。

[Ablet](https://github.com/kaikoga/Ablet-Unity) は同じことをしているので、同等の挙動が望まれる場合はAbletへの移行を推奨します。
